# BGO TOOLKIT

## Project Context

BGO Toolkit adalah aplikasi produktivitas internal yang dibuat untuk membantu aktivitas operasional Bank Garansi (BG).

Aplikasi ini lahir dari kebutuhan sehari-hari pelaksana BGO yang masih sangat bergantung pada:

* Excel
* WhatsApp
* Email
* File lokal

untuk melakukan pencatatan aplikasi, monitoring proses, dan perhitungan provisi.

Saat ini pekerjaan tersebut dilakukan secara manual dan tersebar di berbagai file sehingga menimbulkan:

* Duplikasi pekerjaan
* Kesulitan pencarian data
* Risiko human error
* Ketergantungan pada file Excel tertentu
* Sulit melakukan tracking aplikasi secara cepat

Tujuan aplikasi ini bukan menggantikan sistem bank, melainkan menjadi alat bantu kerja pribadi yang lebih cepat dan efisien.

---

# Target User

Pelaksana BGO.

Karakteristik user:

* Menggunakan Excel setiap hari.
* Terbiasa bekerja dengan keyboard.
* Membutuhkan kecepatan.
* Lebih mengutamakan fungsi dibanding desain.
* Sering melakukan copy-paste data.

---

# Design Philosophy

Prinsip utama aplikasi:

Speed > Beauty

Usability > Complexity

Keyboard First

Offline First

Excel-like Experience

Aplikasi harus terasa seperti:

"Excel yang dibuat khusus untuk pekerjaan BGO"

bukan seperti aplikasi enterprise yang kompleks.

---

# Current Workflow

Pelaksana menerima aplikasi BG dari cabang.

Kemudian melakukan:

1. Review dokumen.
2. Menghitung provisi.
3. Mencatat informasi aplikasi.
4. Monitoring status aplikasi.
5. Menyampaikan hasil ke cabang.

Informasi yang paling sering digunakan:

* Nama Applicant
* Nilai BG
* Nomor BG
* Nomor Registrasi
* Nomor Aplikasi
* Cabang
* Jenis BG
* Nilai Provisi
* Status Proses

Karena itu informasi tersebut menjadi data utama dalam aplikasi.

---

# MVP Goal

MVP dianggap berhasil apabila:

* Pelaksana dapat menghitung provisi tanpa membuka Excel.
* Pelaksana dapat mencatat aplikasi lebih cepat dibanding Excel biasa.
* Pelaksana dapat mencari data aplikasi dalam hitungan detik.
* Pelaksana dapat membuat rekap biaya dengan cepat.
* Seluruh aplikasi berjalan tanpa internet.
